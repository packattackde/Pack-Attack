# Domain Terms — Intentionally Untranslated

These terms appear identically in both the German and English versions of the UI.
This is a **conscious design decision**, not an oversight.

## Brand Names
- **PullForge** — app name
- **PullForge Team** — author attribution

## Game Names (product names, always in their official form)
- **Magic: The Gathering** (MTG)
- **Pokémon**
- **Yu-Gi-Oh!**
- **One Piece**
- **Disney Lorcana**
- **Digimon**
- **Flesh & Blood**

## Game Mechanics / TCG Terminology
These originate from the card games themselves and are internationally recognized:
- **Booster Pack**, **Booster Box**, **Starter Deck**, **Structure Deck**
- **Common**, **Uncommon**, **Rare**, **Epic**, **Legendary**, **Mythic**, **Secret Rare**, **Hyper Rare**
  (rarity labels come from the database and match the official card game terminology)
- **Sealed** (sealed product designation)

## Platform Terms
- **Coins** — in-game currency (used identically in both locales)
- **Battle**, **VS** — competitive mode branding
- **Pack** — card pack unit
- **Bot** — automated test player (admin-only)
- **Level**, **XP** — progression system
- **Dashboard** — user dashboard (kept as English in both locales for UI consistency)

## Third-Party
- **Twitch**, **Discord** — platform names
- **PayPal**, **Stripe** — payment provider names
- **Scryfall**, **PokemonTCG.io**, **JustTCG**, **Lorcana-API** — card data API names (admin-only)

## Format
- **EUR** / **€** — currency symbol (international standard)
- **SKU** — stock keeping unit (industry standard)

---

If a term is NOT listed here but appears untranslated in the UI, it is a bug and should be reported.
